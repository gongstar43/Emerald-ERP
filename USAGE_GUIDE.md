## 📝 كيفية الاستخدام - How to Use

### عروض الأسعار - Quotations

#### إنشاء عرض سعر جديد:
1. انقر على زر "إضافة" في أعلى الصفحة
2. اختر العميل من القائمة المنسدلة
3. حدد التاريخ وتاريخ الصلاحية
4. أضف الأصناف (الوصف، الكمية، السعر)
5. أضف خصم إذا لزم الأمر
6. أدخل الملاحظات
7. انقر "حفظ"

#### تغيير حالة العرض:
- **مسودة → مرسل**: انقر على أيقونة الإرسال (✉️)
- **مرسل → مقبول**: انقر على أيقونة القبول (✓)
- **مرسل → مرفوض**: انقر على أيقونة الرفض (✗)

---

### أوامر البيع - Sales Orders

#### إنشاء أمر بيع:
1. انقر على "إضافة"
2. اختر العميل
3. حدد تاريخ الطلب وتاريخ التسليم المتوقع
4. أضف عنوان الشحن
5. أضف الأصناف مع الكميات والأسعار
6. أضف الخصم والملاحظات
7. احفظ الأمر

#### تتبع حالة الطلب:
```
قيد الانتظار → مؤكد → قيد المعالجة → تم الشحن → تم التسليم
Pending → Confirmed → Processing → Shipped → Delivered
```

---

### أوامر الشراء - Purchase Orders

#### إنشاء أمر شراء:
1. اضغط "إضافة"
2. اختر المورد
3. حدد التاريخ والتسليم المتوقع
4. أدخل عنوان التسليم
5. أضف الأصناف المطلوبة
6. أضف رقم مرجعي (اختياري)
7. احفظ الأمر

#### سير عمل أمر الشراء:
```
قيد الانتظار → معتمد → تم الطلب → تم الاستلام
Pending → Approved → Ordered → Received
```

---

### فواتير الشراء - Purchase Invoices

#### إنشاء فاتورة شراء:
1. انقر "إضافة"
2. اختر المورد
3. حدد تاريخ الفاتورة وتاريخ الاستحقاق
4. أدخل رقم أمر الشراء (إن وُجد)
5. أضف الأصناف
6. أضف خصم إذا لزم
7. احفظ الفاتورة

#### تسجيل دفعة:
1. اضغط على أيقونة الدولار ($) في صف الفاتورة
2. أدخل مبلغ الدفعة
3. اضغط "تأكيد"
4. يتم تحديث الحالة تلقائياً:
   - **مدفوعة جزئياً**: إذا كان المبلغ أقل من الإجمالي
   - **مدفوعة**: إذا تم دفع المبلغ كاملاً

---

### المستودعات - Warehouses

#### إضافة مستودع جديد:
1. اضغط "إضافة"
2. أدخل:
   - الكود (مثل: WH-001)
   - الاسم
   - الموقع
   - العنوان الكامل
   - السعة (بالقطع)
   - اسم المدير
   - رقم الهاتف
   - البريد الإلكتروني
3. أضف ملاحظات
4. احفظ

#### عرض تفاصيل المستودع:
- انقر على زر "عرض" في بطاقة المستودع
- شاهد:
  - معلومات المستودع
  - نسبة الإشغال
  - قائمة الأصناف المخزنة
  - مواقع الأصناف داخل المستودع

---

### حركات المخزون - Movements

#### إضافة حركة وارد (In):
1. اضغط "إضافة"
2. اختر نوع الحركة: **وارد**
3. أدخل:
   - كود الصنف
   - اسم الصنف
   - الكمية
   - الوحدة
   - إلى مستودع
   - المرجع (مثل: PO-2024-001)
4. أضف ملاحظات
5. احفظ

#### إضافة حركة صادر (Out):
1. نفس الخطوات
2. اختر نوع الحركة: **صادر**
3. حدد **من مستودع**
4. لا تحدد "إلى مستودع"

#### إضافة حركة تحويل (Transfer):
1. نفس الخطوات
2. اختر نوع الحركة: **تحويل**
3. حدد كلاً من:
   - من مستودع
   - إلى مستودع

#### الإحصائيات:
- شاهد في أعلى الصفحة:
  - إجمالي الوارد (أخضر)
  - إجمالي الصادر (أحمر)
  - إجمالي التحويلات (أزرق)

## 💡 نصائح مهمة - Important Tips

### 1. الحسابات التلقائية:
- ✅ يتم حساب المجموع لكل صنف تلقائياً (الكمية × السعر)
- ✅ يتم حساب الضريبة تلقائياً (15%)
- ✅ يتم خصم الخصم من الإجمالي
- ✅ يظهر الإجمالي النهائي فوراً

### 2. التواريخ:
- تاريخ اليوم يُملأ تلقائياً
- يمكن تغيير التواريخ حسب الحاجة
- تاريخ الصلاحية/التسليم يُحسب تلقائياً (30 يوم للعروض، 14 يوم للأوامر)

### 3. الحالات:
- جميع السجلات تبدأ كـ "مسودة" أو "قيد الانتظار"
- استخدم الأزرار السريعة لتغيير الحالة
- لا يمكن تعديل السجلات المكتملة (يجب الحذف وإعادة الإنشاء)

### 4. العملاء والموردين:
- القوائم المنسدلة تحتوي على بيانات تجريبية
- في النظام الحقيقي، ستُربط بقاعدة البيانات
- يمكن إضافة المزيد من العملاء/الموردين من صفحاتهم المخصصة

### 5. المستودعات:
- مؤشر السعة يتغير لونه:
  - **أخضر**: أقل من 60%
  - **أصفر**: 60-80%
  - **أحمر**: أكثر من 80%

## 🔄 سير العمل الموصى به - Recommended Workflow

### للمبيعات:
```
1. إنشاء عرض سعر → 
2. إرسال للعميل → 
3. قبول العرض → 
4. إنشاء أمر بيع → 
5. تأكيد الطلب → 
6. معالجة وشحن → 
7. تسليم للعميل → 
8. إنشاء فاتورة بيع
```

### للمشتريات:
```
1. إنشاء أمر شراء → 
2. اعتماد الأمر → 
3. إرسال للمورد → 
4. استلام البضاعة → 
5. إنشاء فاتورة شراء → 
6. تسجيل الدفعات
```

### للمخزون:
```
1. إنشاء المستودعات → 
2. تسجيل الأصناف → 
3. استلام بضاعة (حركة وارد) → 
4. صرف للعملاء (حركة صادر) → 
5. نقل بين المستودعات (تحويل) → 
6. مراجعة الإحصائيات
```
