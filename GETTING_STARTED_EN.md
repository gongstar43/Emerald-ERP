# 🚀 Quick Start Guide - ERP System

## Welcome to the Comprehensive ERP System!

This system provides you with an integrated platform to manage all aspects of your business: accounting, sales, purchases, inventory, and human resources.

---

## 📋 First Steps

### 1️⃣ Create a New Account

**⭐ VERY IMPORTANT NOTE:**
- **The first user to sign up in the system automatically gets Admin privileges**
- Subsequent users get regular User privileges

**Steps to Create Admin Account:**

1. Open the application - you'll see the login page
2. Click on **"Don't have an account?"**
3. Enter:
   - **Full Name**: Your name or username (e.g., "System Admin")
   - **Email**: Any valid email address (e.g., admin@company.com)
   - **Password**: Choose a strong password (e.g., Admin@123)
4. Click **"Sign Up"**
5. You'll be automatically logged in **with full Admin privileges** ✅

### 2️⃣ Explore the Dashboard

After logging in, you'll find:

- **📊 Quick Statistics**: Invoices, Payments, Contacts
- **📈 Recent Activity**: Latest system operations
- **⚡ Quick Actions**: Shortcuts to common operations

### 3️⃣ Navigate the System

Use the **Sidebar Menu** to access:

#### 🏠 Main Pages
- **Dashboard**: Comprehensive system overview
- **Contacts**: Manage customers and suppliers
- **Businesses**: Manage companies and organizations

#### 💰 Accounting

## 🌍 Change Language

You can switch between **Arabic** and **English** anytime:

1. Click the **🌐 Globe** button in:
   - Top of screen (login page)
   - User dropdown menu (after login)
2. Language will switch instantly with text direction (RTL/LTR)

## 👥 Add a New Contact

### Quick Method:
1. Go to **"Contacts"** from the sidebar
2. Click the **"+ Add"** button
3. Fill in the details:
   - Name
   - Email
   - Phone
   - Type (Customer/Supplier/Both)
   - Address (optional)
4. Click **"Save"**

### Additional Features:
- **🔍 Search**: Search contacts by name, email, or phone
- **✏️ Edit**: Click the pencil icon to edit any contact
- **🗑️ Delete**: Click the trash icon to delete (with confirmation)

## 📊 Understanding the Dashboard

### Statistics Cards:
1. **Total Invoices**: Total number of invoices (pending + paid)
