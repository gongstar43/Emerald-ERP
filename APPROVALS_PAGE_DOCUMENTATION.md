## ✨ المميزات الرئيسية - Key Features

### 1. لوحة إحصائيات شاملة
- **المعلقة (Pending):** عدد الطلبات المنتظرة للموافقة
- **الموافق عليها (Approved):** عدد الطلبات المعتمدة
- **المرفوضة (Rejected):** عدد الطلبات المرفوضة
- **العاجلة (Urgent):** عدد الطلبات العاجلة المعلقة

### 2. أنواع الطلبات المدعومة
- 📅 **الإجازات (Leave):** طلبات الإجازات السنوية والطارئة
- 💰 **المصروفات (Expenses):** مصروفات السفر والأعمال
- 🛒 **المشتريات (Purchases):** طلبات ��راء المعدات والمواد
- 💼 **المشاريع (Projects):** اعتماد المشاريع الجديدة
- 💵 **المدفوعات (Payments):** دفعات الموردين والفواتير

### 3. مستويات الأولوية
- 🔴 **عاجلة (Urgent):** تتطلب إجراء فوري
- 🟠 **عالية (High):** أولوية عالية
- 🔵 **متوسطة (Medium):** أولوية متوسطة
- ⚫ **منخفضة (Low):** أولوية منخفضة

### 4. نظام تصفية متقدم
- 🔍 **البحث النصي:** بحث في العنوان، مقدم الطلب، أو رقم الطلب
- 📁 **تصفية حسب النوع:** إجازة، مصروف، شراء، مشروع، دفعة
- 🎯 **تصفية حسب الأولوية:** عاجلة، عالية، متوسطة، منخفضة
- 📊 **تبويبات الحالة:** الكل، معلقة، موافق عليها، مرفوضة

### 5. عرض تفصيلي للطلبات
- معلومات كاملة عن الطلب
- تفاصيل مقدم الطلب والقسم
- المبلغ (إن وجد) بالعملة المحلية
- تاريخ الطلب والموعد النهائي
- حساب الأيام المتبقية تلقائياً
- مستوى الموافقة الحالي (Progress Bar)
- وصف تفصيلي للطلب

### 6. إجراءات الموافقة
- ✅ **موافقة سريعة:** موافقة مباشرة بنقرة واحدة
- ❌ **رفض مع ملاحظات:** رفض مع إمكانية كتابة سبب الرفض
- 👁️ **عرض التفاصيل:** فتح نافذة منبثقة بكامل التفاصيل

### 7. تنبيهات ذكية
- 🔔 تنبيه عند اقتراب الموعد النهائي
- ⚠️ تمييز الطلبات المتأخرة بلون أحمر
- ✅ رسائل نجاح عند الموافقة أو الرفض

## 💻 الكود البرمجي - Code Structure

### Interface الرئيسي
```typescript
interface ApprovalRequest {
  id: string;
  type: 'leave' | 'expense' | 'purchase' | 'project' | 'payment';
  title: string;
  requester: string;
  department: string;
  amount?: number;
  currency?: string;
  priority: 'low' | 'medium' | 'high' | 'urgent';
  status: 'pending' | 'approved' | 'rejected';
  requestDate: string;
  dueDate: string;
  description: string;
  approvalLevel: number;
  totalLevels: number;
}
```

### Functions الرئيسية
- `handleApprove(id)`: معالجة الموافقة
- `handleReject(id)`: معالجة الرفض مع الملاحظات
- `handleViewDetails(approval)`: عرض التفاصيل الكاملة
- `getDaysRemaining(dueDate)`: حساب الأيام المتبقية
- `formatCurrency(amount, currency)`: تنسيق المبلغ
